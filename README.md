# LOLGEN: Living Off The Land Payload Generator

Online: [https://lolgen.hdks.org/](https://lolgen.hdks.org/)

## Use LOLGEN in Local

LOLGEN is also available in your local environment:

```sh
git clone https://github.com/hideckies/LOLGEN.git
cd LOLGEN
npm install
npm run dev
```
