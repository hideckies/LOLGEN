LOLGEN: LIVING OFF THE LAND PAYLOAD GENERATOR

LOLGEN is inspired by MITRE ATT&CK and LOLBAS. The source code is HERE.

The first generated payload is a simple command, and the second is an obfuscated version of the first. Both are editable and can be executed on the target Windows system by coping and pasting.
